# ATA 92 — MODEL BASED MAINTENANCE

## Overview

This is ATA Chapter 92: MODEL BASED MAINTENANCE, part of the T — TECHNOLOGY AMEDEOPELLICCIA — ON BOARD SYSTEMS axis.

## Structure

This chapter follows the mandatory OPT-IN Framework structure:

The GENERAL layer contains 14 mandatory lifecycle folders covering the complete development cycle:

1. **92-00-01_Overview**: System overview and global architecture
2. **92-00-02_Safety**: Safety framework and analysis
3. **92-00-03_Requirements**: Requirements and traceability
4. **92-00-04_Design**: Design specifications and patterns
5. **92-00-05_Interfaces**: Interface control documents
6. **92-00-06_Engineering**: Analysis, models, and simulation
7. **92-00-07_V_AND_V**: Verification and validation
8. **92-00-08_Prototyping**: Prototype development
9. **92-00-09_Production_Planning**: Manufacturing planning
10. **92-00-10_Certification**: Certification evidence
11. **92-00-11_EIS_Versions_Tags**: Configuration management
12. **92-00-12_Services**: Maintenance and service
13. **92-00-13_Subsystems_Components**: Component breakdown
14. **92-00-14_Ops_Std_Sustain**: Operational standards
