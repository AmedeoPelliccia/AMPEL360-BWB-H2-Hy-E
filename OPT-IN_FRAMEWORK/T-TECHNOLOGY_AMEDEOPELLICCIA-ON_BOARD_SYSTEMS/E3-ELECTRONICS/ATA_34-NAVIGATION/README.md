# ATA 34 — NAVIGATION

## Overview

This is ATA Chapter 34: NAVIGATION, part of the T — TECHNOLOGY AMEDEOPELLICCIA — ON BOARD SYSTEMS axis.

## Structure

This chapter follows the mandatory OPT-IN Framework structure:

### 34-00_GENERAL (Lifecycle Folders)

The GENERAL layer contains 14 mandatory lifecycle folders covering the complete development cycle:

1. **34-00-01_Overview**: System overview and global architecture
2. **34-00-02_Safety**: Safety framework and analysis
3. **34-00-03_Requirements**: Requirements and traceability
4. **34-00-04_Design**: Design specifications and patterns
5. **34-00-05_Interfaces**: Interface control documents
6. **34-00-06_Engineering**: Analysis, models, and simulation
7. **34-00-07_V_AND_V**: Verification and validation
8. **34-00-08_Prototyping**: Prototype development
9. **34-00-09_Production_Planning**: Manufacturing planning
10. **34-00-10_Certification**: Certification evidence
11. **34-00-11_EIS_Versions_Tags**: Configuration management
12. **34-00-12_Services**: Maintenance and service
13. **34-00-13_Subsystems_Components**: Component breakdown
14. **34-00-14_Ops_Std_Sustain**: Operational standards

### Cross-ATA Root Buckets

The following buckets are mandatory in every ATA chapter:

- **34-10_Operations**: Operational procedures and use cases
- **34-20_Subsystems**: Functional subsystems (design-driven internal structure)
- **34-30_Circularity**: Sustainability, LCA, and circular economy
- **34-40_Software**: Software, control logic, and AI/ML
- **34-50_Structures**: Physical structures and frames
- **34-60_Storages**: Tanks, reservoirs, and storage
- **34-70_Propulsion**: Propulsion interfaces (if applicable)
- **34-80_Energy**: Energy management and distribution
- **34-90_Tables_Schemas_Diagrams**: Data tables and documentation
