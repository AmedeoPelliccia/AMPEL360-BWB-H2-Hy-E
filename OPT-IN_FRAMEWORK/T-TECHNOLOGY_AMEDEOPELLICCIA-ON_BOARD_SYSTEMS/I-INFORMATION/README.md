## Overview

The I-INFORMATION domain encompasses all information management, display, and interface systems on board the aircraft, including flight deck displays, maintenance systems, and data management platforms.

## Scope

Information systems include:
- **Display Systems**: Primary flight displays, MFDs
- **Crew Interfaces**: Human-machine interfaces
- **IMA Core Services**: Operating system services
- **Maintenance Systems**: Health monitoring, diagnostics
- **Information Management**: Data processing and storage
- **Engine Indicating**: Engine parameter displays
