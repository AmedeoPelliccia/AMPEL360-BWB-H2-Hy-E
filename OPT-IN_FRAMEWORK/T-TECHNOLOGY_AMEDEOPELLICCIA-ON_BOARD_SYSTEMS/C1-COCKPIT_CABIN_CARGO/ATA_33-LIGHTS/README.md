# ATA 33 — LIGHTS

## Overview

This is ATA Chapter 33: LIGHTS, part of the T — TECHNOLOGY AMEDEOPELLICCIA — ON BOARD SYSTEMS axis.

## Structure

This chapter follows the mandatory OPT-IN Framework structure:

### 33-00_GENERAL (Lifecycle Folders)

The GENERAL layer contains 14 mandatory lifecycle folders covering the complete development cycle:

1. **33-00-01_Overview**: System overview and global architecture
2. **33-00-02_Safety**: Safety framework and analysis
3. **33-00-03_Requirements**: Requirements and traceability
4. **33-00-04_Design**: Design specifications and patterns
5. **33-00-05_Interfaces**: Interface control documents
6. **33-00-06_Engineering**: Analysis, models, and simulation
7. **33-00-07_V_AND_V**: Verification and validation
8. **33-00-08_Prototyping**: Prototype development
9. **33-00-09_Production_Planning**: Manufacturing planning
10. **33-00-10_Certification**: Certification evidence
11. **33-00-11_EIS_Versions_Tags**: Configuration management
12. **33-00-12_Services**: Maintenance and service
13. **33-00-13_Subsystems_Components**: Component breakdown
14. **33-00-14_Ops_Std_Sustain**: Operational standards

### Cross-ATA Root Buckets

The following buckets are mandatory in every ATA chapter:

- **33-10_Operations**: Operational procedures and use cases
- **33-20_Subsystems**: Functional subsystems (design-driven internal structure)
- **33-30_Circularity**: Sustainability, LCA, and circular economy
- **33-40_Software**: Software, control logic, and AI/ML
- **33-50_Structures**: Physical structures and frames
- **33-60_Storages**: Tanks, reservoirs, and storage
- **33-70_Propulsion**: Propulsion interfaces (if applicable)
- **33-80_Energy**: Energy management and distribution
- **33-90_Tables_Schemas_Diagrams**: Data tables and documentation
