The internal structure of this bucket is **design-driven** and flexible:

## Naming Convention

Items within this bucket follow the pattern:
- **11-40-XX_DESCRIPTION**
  - 11 = ATA chapter
  - 40 = Bucket number
  - XX = Sequential number (00, 01, 02, etc.)
  - DESCRIPTION = Descriptive name
