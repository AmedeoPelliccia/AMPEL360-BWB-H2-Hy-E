## Current Configuration

The baseline cabin configuration for the Q100 variant accommodates **120 passengers** in a single-class layout optimized for the BWB architecture.

### Key Specifications
- **Passengers:** 120
- **Seating:** Mixed 2-3-2 and 2-2-2 configuration
- **Cabin Length:** ~19.24 meters
- **Cabin Width:** 4.55 meters (midsection)
- **Dual Aisles:** 2 × 500 mm
- **Galleys:** Forward (full-service) + Aft (compact)
- **Lavatories:** 4 units
- **Emergency Exits:** 8 (4 port/4 starboard)

## BWB Configuration Advantages

The AMPEL360 BWB design provides unique benefits for cabin configuration:
1. **Wide Cabin:** 4,550 mm width enables dual-aisle configuration in 120-seat aircraft
2. **Passenger Comfort:** 71% more width per passenger vs. conventional narrow-body
3. **Volume Efficiency:** 2.67 m³ per passenger (wide-body comfort level)
4. **Safety:** Multiple emergency exits with 50%+ capacity margin
5. **Operational Efficiency:** Dual galleys and aisles support fast turnaround
