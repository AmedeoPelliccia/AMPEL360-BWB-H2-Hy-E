# ATA 45 — ONBOARD MAINTENANCE SYSTEMS

## Overview

This is ATA Chapter 45: ONBOARD MAINTENANCE SYSTEMS, part of the T — TECHNOLOGY AMEDEOPELLICCIA — ON BOARD SYSTEMS axis.

## Structure

This chapter follows the mandatory OPT-IN Framework structure:

### 45-00_GENERAL (Lifecycle Folders)

The GENERAL layer contains 14 mandatory lifecycle folders covering the complete development cycle:

1. **45-00-01_Overview**: System overview and global architecture
2. **45-00-02_Safety**: Safety framework and analysis
3. **45-00-03_Requirements**: Requirements and traceability
4. **45-00-04_Design**: Design specifications and patterns
5. **45-00-05_Interfaces**: Interface control documents
6. **45-00-06_Engineering**: Analysis, models, and simulation
7. **45-00-07_V_AND_V**: Verification and validation
8. **45-00-08_Prototyping**: Prototype development
9. **45-00-09_Production_Planning**: Manufacturing planning
10. **45-00-10_Certification**: Certification evidence
11. **45-00-11_EIS_Versions_Tags**: Configuration management
12. **45-00-12_Services**: Maintenance and service
13. **45-00-13_Subsystems_Components**: Component breakdown
14. **45-00-14_Ops_Std_Sustain**: Operational standards

### Cross-ATA Root Buckets

The following buckets are mandatory in every ATA chapter:

- **45-10_Operations**: Operational procedures and use cases
- **45-20_Subsystems**: Functional subsystems (design-driven internal structure)
- **45-30_Circularity**: Sustainability, LCA, and circular economy
- **45-40_Software**: Software, control logic, and AI/ML
- **45-50_Structures**: Physical structures and frames
- **45-60_Storages**: Tanks, reservoirs, and storage
- **45-70_Propulsion**: Propulsion interfaces (if applicable)
- **45-80_Energy**: Energy management and distribution
- **45-90_Tables_Schemas_Diagrams**: Data tables and documentation
