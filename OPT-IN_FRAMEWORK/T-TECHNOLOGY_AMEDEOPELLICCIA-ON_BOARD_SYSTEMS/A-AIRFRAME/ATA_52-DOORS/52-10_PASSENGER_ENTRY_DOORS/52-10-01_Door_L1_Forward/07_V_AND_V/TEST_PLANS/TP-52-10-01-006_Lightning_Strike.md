# Test Plan - Lightning Strike Test
## Door L1 Forward

**Test Plan:** TP-52-10-01-006  
**Priority:** MEDIUM  
**Revision:** 1.0  
**Date:** 2025-11-03  
**Status:** Draft

---

## 1. TEST OBJECTIVES

Demonstrate lightning strike survivability per CS-25.581 for Zone 1A (direct strike) with 200 kA peak current.

---

## 2. TEST REQUIREMENTS

### 2.1 Lightning Environment
- **Zone:** 1A (direct strike, high probability)
- **Peak current:** 200 kA
- **Waveform:** Component A (0/350 μs) + Component D (continuing current)
- **Action integral:** >2 × 10⁶ A²s

### 2.2 Acceptance Criteria
**PASS:**
- No structural damage affecting airworthiness
- No puncture or burn-through
- Safe current path established
- Damage repairable if occurs

---

## 3. TEST SETUP

### 3.1 Test Article
- Door panel with lightning protection system
- Diverter strips installed
- Grounding straps connected
- Bonding verified

### 3.2 Test Facility
- High-voltage laboratory
- Lightning simulator (200 kA capability)
- Attachment points per SAE ARP5412B

---

## 4. TEST PROCEDURE

### 4.1 Pre-Test
1. Install lightning protection features
2. Verify electrical continuity
3. Document baseline condition
4. Position test article

### 4.2 Lightning Strike
1. Apply Component A waveform (200 kA)
2. Apply Component D (continuing current)
3. Record current path
4. Document arc attachment points

### 4.3 Post-Test Inspection
1. Visual inspection for damage
2. NDT inspection (C-scan)
3. Measure burn areas
4. Assess repairability

---
