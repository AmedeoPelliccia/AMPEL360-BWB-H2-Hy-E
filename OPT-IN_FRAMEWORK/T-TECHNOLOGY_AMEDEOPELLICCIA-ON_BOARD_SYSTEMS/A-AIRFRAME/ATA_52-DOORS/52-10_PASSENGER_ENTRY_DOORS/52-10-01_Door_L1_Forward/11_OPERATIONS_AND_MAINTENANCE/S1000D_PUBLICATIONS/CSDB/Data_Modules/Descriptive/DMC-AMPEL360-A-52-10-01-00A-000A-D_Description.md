# Door L1 Forward - System Description
## DMC-AMPEL360-A-52-10-01-00A-000A-D

---

## 1. GENERAL DESCRIPTION

The Door L1 Forward is a Type A passenger entry door located on the left forward fuselage of the AMPEL360 BWB H2 Hy-E aircraft. This door serves as the primary passenger boarding and deplaning access point.

### 1.1 Door Designation
- **ATA Chapter:** 52-10-01
- **Location:** Left side, forward section
- **Type:** Type A Passenger Entry Door
- **Classification:** Emergency Exit Capable

### 1.2 Primary Functions
- Normal passenger boarding and deplaning
- Emergency evacuation (required for certification)
- Pressurization boundary integrity
- Structural continuity of fuselage
- Integration with passenger boarding bridge

---

## 2. SYSTEM OVERVIEW

### 2.1 Door Assembly
The door assembly consists of:
- Door structure (composite construction)
- Inner and outer door panels
- Pressure seal system
- Latching mechanism (8-point latch)
- Hinge assemblies (2 main hinges)
- Window assembly
- Emergency release handle
- Power-assist actuator

### 2.2 Dimensions
- **Height:** 1.83 m (72 inches)
- **Width:** 1.07 m (42 inches)
- **Weight:** 68 kg (150 lbs) including hardware
- **Thickness:** 85 mm (3.35 inches)

### 2.3 Materials
- **Structure:** Carbon fiber reinforced polymer (CFRP)
- **Seals:** Silicone rubber (high-temperature resistant)
- **Window:** Multilayer polycarbonate/acrylic laminate
- **Hardware:** Titanium alloys (Grade 5, Ti-6Al-4V)

## 3. OPERATIONAL MODES

### 3.1 Normal Operation
- **Opening:** Power-assisted with manual backup
- **Closing:** Manual with power-assist option
- **Locking:** Automatic engagement when closed
- **Indication:** Door status displayed on flight deck and cabin

### 3.2 Emergency Operation
- **Internal Release:** Single-action handle operation
- **External Access:** Tool-free emergency access panel
- **Time to Open:** < 5 seconds from initiation
- **Emergency Slide:** Automatic deployment when armed

### 3.3 Ground Service
- **Maintenance Mode:** Manual override for ground operations
- **Adjustment Mode:** Rigging and alignment capability
- **Test Mode:** Functional verification without flight systems active
