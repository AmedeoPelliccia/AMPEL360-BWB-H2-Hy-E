## 1. INTERFACE OVERVIEW

This ICD reserves interface provisions for future hydraulic power assist actuation.

---

## 2. REQUIREMENTS (CONCEPTUAL)

### 2.1 Hydraulic System Interface
- System: Green hydraulic system (3000 psi)
- Flow rate: 2 GPM maximum
- Actuator: Linear, 500mm stroke
- Force: 5 kN maximum

### 2.2 Connection Points
- Supply line: 1/2" port at FS 105
- Return line: 1/2" port at FS 105
- Shutoff valve: Electrically controlled

---

## 3. STATUS

Current design uses electric motor actuation. Hydraulic interface reserved for:
- Heavy door variants
- High-cycle applications
- Emergency backup system

---

## 4. IMPLEMENTATION

Implementation requires:
- Detailed hydraulic system analysis
- Actuator selection and sizing
- Plumbing design and routing
- Emergency power-off provisions
