# Quality Control Procedures
## Manufacturing Inspection Plan

**Document:** MFG-DEV-QC-001  
**Status:** Development  
**Date:** 2025-11-03

### In-Process Inspections

#### 1. Material Receiving
- [ ] Material certification review
- [ ] Storage temperature log
- [ ] Shelf life verification
- [ ] Visual inspection

#### 2. Layup Inspections
- [ ] Tool cleanliness
- [ ] Ply orientation (±1°)
- [ ] Overlap dimensions (3-6mm)
- [ ] Foreign object debris (FOD) check
- [ ] Photo documentation

#### 3. Cure Process
- [ ] Bagging integrity (leak check)
- [ ] Thermocouple placement
- [ ] Cure cycle chart review
- [ ] Temperature uniformity

#### 4. Post-Cure Inspections
- [ ] Visual (surface defects)
- [ ] Dimensional (CMM or templates)
- [ ] Ultrasonic C-scan (void content)
- [ ] Weight verification

### Non-Destructive Testing (NDT)

**Ultrasonic C-Scan:**
- **Method:** Through-transmission
- **Frequency:** 5 MHz
- **Grid:** 2mm resolution
- **Acceptance:** <5% void content
- **Documentation:** Digital scan files

**Tap Testing:**
- Manual inspection
- Identify delaminations
- 100% coverage
- Document anomalies

### Acceptance Criteria

| Parameter | Specification | Measure Method |
|-----------|---------------|----------------|
| Thickness | 48±1mm | Micrometer (10 pts) |
| Weight | 22±2kg | Scale |
| Flatness | <2mm | Straightedge |
| Voids | <5% area | C-scan |
| Surface | A-quality | Visual |
| Dimensional | ±2mm | CMM or template |

### Documentation

Each article requires:
1. Material certifications
2. Layup photos
3. Cure cycle chart
4. C-scan report
5. Dimensional inspection
6. Final acceptance signoff
