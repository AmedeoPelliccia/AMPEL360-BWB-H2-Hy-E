# Cure Tool Specification
## Autoclave Tooling

**Document:** TOOL-DEV-CURE-001  
**Status:** Conceptual  
**Date:** 2025-11-03

### Tool Set Components

1. **Cure Mold** (primary tool)
   - Female mold for OML surface
   - See Mold_Design.md

2. **Caul Plates**
   - Aluminum or steel
   - Pressure distribution
   - Corner radius forming
   - Quantity: 3 (flat sections + corners)

3. **Edge Dams**
   - Control resin bleed
   - Maintain edge thickness
   - Silicone rubber construction

4. **Vacuum Bag Consumables**
   - Peel ply (A4000 Green)
   - Breather (Airweave N10)
   - Release film (FEP perforated)
   - Vacuum bag (nylon 6)
   - Sealant tape

### Autoclave Requirements

**Size:** Minimum 2.5m × 1.5m × 1.5m  
**Pressure:** 0-10 bar capability  
**Temperature:** 0-200°C  
**Vacuum:** 28 inHg minimum

### Tooling Maintenance

- Clean after each use
- Re-release agent every 3 uses
- Dimensional inspection (quarterly)
- Surface refinishing (as needed)
