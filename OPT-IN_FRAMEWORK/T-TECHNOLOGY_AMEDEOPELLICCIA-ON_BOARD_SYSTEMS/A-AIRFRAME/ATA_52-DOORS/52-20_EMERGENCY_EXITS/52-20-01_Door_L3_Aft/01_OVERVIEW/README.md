## Key Differentiators from L1 Forward

- **Emergency Exit Type:** Type A emergency exit (not regular entry)
- **Location:** Aft position in BWB configuration
- **Slide Pack:** Dual-lane slide for rapid evacuation
- **Opening:** Power-assist with manual override priority
- **CAOS Integration:** Emergency monitoring priority
