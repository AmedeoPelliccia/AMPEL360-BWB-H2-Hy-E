# Assembly Sequence - Pressure Shell Modules

## Purpose
Defines the assembly sequence for pressure shell module fabrication and integration.

## Major Assembly Steps

### Phase 1: Component Fabrication (Months 1-6)
1. Upper shell panel fabrication
2. Lower shell panel fabrication
3. Side shell panel fabrication
4. Forward bulkhead fabrication
5. Aft bulkhead fabrication

### Phase 2: Section Assembly (Months 7-9)
1. Forward section assembly (FS 0-15)
2. Center section assembly (FS 15-30)
3. Aft section assembly (FS 30-38.5)

### Phase 3: Final Integration (Months 10-12)
1. Circumferential splice assembly
2. Bulkhead installation
3. Systems installation
4. Pressure testing
5. Final inspection

See detailed procedures in manufacturing manual.
