# 53-20-A-202 Fastener Standards

## Purpose

Defines approved fasteners, installation methods, and torque specifications for ATA 53-20 Subsystems.

## Approved Fasteners

### Hi-Lok Fasteners
- **Material**: Titanium Ti-6Al-4V per AMS 4967
- **Sizes**: 1/4" (6.35mm) through 3/8" (9.53mm) diameter
- **Installation**: Pin + collar assembly, non-removable
- **Torque**: Per manufacturer specification (typically 45-65 in-lb)

### Interference Fit Requirements
- **Interference**: 0.13-0.25mm (0.005-0.010 inches)
- **Application**: Fatigue-critical joints

## Installation Requirements

All fastener installation must comply with:
- [CS-25.607](https://www.easa.europa.eu/en/document-library/certification-specifications/cs-25-large-aeroplanes) - Fasteners
- Manufacturer installation procedures
- NDT inspection after installation
