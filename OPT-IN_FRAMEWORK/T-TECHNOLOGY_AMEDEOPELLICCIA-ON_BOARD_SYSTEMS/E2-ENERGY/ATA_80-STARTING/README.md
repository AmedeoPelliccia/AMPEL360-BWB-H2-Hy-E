# ATA 80 — STARTING

## Overview

This is ATA Chapter 80: STARTING, part of the T — TECHNOLOGY AMEDEOPELLICCIA — ON BOARD SYSTEMS axis.

## Structure

This chapter follows the mandatory OPT-IN Framework structure:

### 80-00_GENERAL (Lifecycle Folders)

The GENERAL layer contains 14 mandatory lifecycle folders covering the complete development cycle:

1. **80-00-01_Overview**: System overview and global architecture
2. **80-00-02_Safety**: Safety framework and analysis
3. **80-00-03_Requirements**: Requirements and traceability
4. **80-00-04_Design**: Design specifications and patterns
5. **80-00-05_Interfaces**: Interface control documents
6. **80-00-06_Engineering**: Analysis, models, and simulation
7. **80-00-07_V_AND_V**: Verification and validation
8. **80-00-08_Prototyping**: Prototype development
9. **80-00-09_Production_Planning**: Manufacturing planning
10. **80-00-10_Certification**: Certification evidence
11. **80-00-11_EIS_Versions_Tags**: Configuration management
12. **80-00-12_Services**: Maintenance and service
13. **80-00-13_Subsystems_Components**: Component breakdown
14. **80-00-14_Ops_Std_Sustain**: Operational standards

### Cross-ATA Root Buckets

The following buckets are mandatory in every ATA chapter:

- **80-10_Operations**: Operational procedures and use cases
- **80-20_Subsystems**: Functional subsystems (design-driven internal structure)
- **80-30_Circularity**: Sustainability, LCA, and circular economy
- **80-40_Software**: Software, control logic, and AI/ML
- **80-50_Structures**: Physical structures and frames
- **80-60_Storages**: Tanks, reservoirs, and storage
- **80-70_Propulsion**: Propulsion interfaces (if applicable)
- **80-80_Energy**: Energy management and distribution
- **80-90_Tables_Schemas_Diagrams**: Data tables and documentation
