# ATA 75 — AIR

## Overview

This is ATA Chapter 75: AIR, part of the T — TECHNOLOGY AMEDEOPELLICCIA — ON BOARD SYSTEMS axis.

## Structure

This chapter follows the mandatory OPT-IN Framework structure:

### 75-00_GENERAL (Lifecycle Folders)

The GENERAL layer contains 14 mandatory lifecycle folders covering the complete development cycle:

1. **75-00-01_Overview**: System overview and global architecture
2. **75-00-02_Safety**: Safety framework and analysis
3. **75-00-03_Requirements**: Requirements and traceability
4. **75-00-04_Design**: Design specifications and patterns
5. **75-00-05_Interfaces**: Interface control documents
6. **75-00-06_Engineering**: Analysis, models, and simulation
7. **75-00-07_V_AND_V**: Verification and validation
8. **75-00-08_Prototyping**: Prototype development
9. **75-00-09_Production_Planning**: Manufacturing planning
10. **75-00-10_Certification**: Certification evidence
11. **75-00-11_EIS_Versions_Tags**: Configuration management
12. **75-00-12_Services**: Maintenance and service
13. **75-00-13_Subsystems_Components**: Component breakdown
14. **75-00-14_Ops_Std_Sustain**: Operational standards

### Cross-ATA Root Buckets

The following buckets are mandatory in every ATA chapter:

- **75-10_Operations**: Operational procedures and use cases
- **75-20_Subsystems**: Functional subsystems (design-driven internal structure)
- **75-30_Circularity**: Sustainability, LCA, and circular economy
- **75-40_Software**: Software, control logic, and AI/ML
- **75-50_Structures**: Physical structures and frames
- **75-60_Storages**: Tanks, reservoirs, and storage
- **75-70_Propulsion**: Propulsion interfaces (if applicable)
- **75-80_Energy**: Energy management and distribution
- **75-90_Tables_Schemas_Diagrams**: Data tables and documentation
