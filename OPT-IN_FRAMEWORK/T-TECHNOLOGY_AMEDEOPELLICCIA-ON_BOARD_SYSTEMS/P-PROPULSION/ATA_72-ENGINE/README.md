# ATA 72 — ENGINE

## Overview

This is ATA Chapter 72: ENGINE, part of the T — TECHNOLOGY AMEDEOPELLICCIA — ON BOARD SYSTEMS axis.

## Structure

This chapter follows the mandatory OPT-IN Framework structure:

### 72-00_GENERAL (Lifecycle Folders)

The GENERAL layer contains 14 mandatory lifecycle folders covering the complete development cycle:

1. **72-00-01_Overview**: System overview and global architecture
2. **72-00-02_Safety**: Safety framework and analysis
3. **72-00-03_Requirements**: Requirements and traceability
4. **72-00-04_Design**: Design specifications and patterns
5. **72-00-05_Interfaces**: Interface control documents
6. **72-00-06_Engineering**: Analysis, models, and simulation
7. **72-00-07_V_AND_V**: Verification and validation
8. **72-00-08_Prototyping**: Prototype development
9. **72-00-09_Production_Planning**: Manufacturing planning
10. **72-00-10_Certification**: Certification evidence
11. **72-00-11_EIS_Versions_Tags**: Configuration management
12. **72-00-12_Services**: Maintenance and service
13. **72-00-13_Subsystems_Components**: Component breakdown
14. **72-00-14_Ops_Std_Sustain**: Operational standards

### Cross-ATA Root Buckets

The following buckets are mandatory in every ATA chapter:

- **72-10_Operations**: Operational procedures and use cases
- **72-20_Subsystems**: Functional subsystems (design-driven internal structure)
- **72-30_Circularity**: Sustainability, LCA, and circular economy
- **72-40_Software**: Software, control logic, and AI/ML
- **72-50_Structures**: Physical structures and frames
- **72-60_Storages**: Tanks, reservoirs, and storage
- **72-70_Propulsion**: Propulsion interfaces (if applicable)
- **72-80_Energy**: Energy management and distribution
- **72-90_Tables_Schemas_Diagrams**: Data tables and documentation
