## Overview

The L2-LINKS domain encompasses all communication and network systems on board the aircraft, including voice communications, data links, IMA network fabric, and operational documentation systems.

## Scope

Link systems include:
- **Voice Communications**: Radio, intercom, PA systems
- **Data Communications**: Datalink, ACARS, satellite
- **IMA Network**: AFDX/ARINC 664 network
- **Network Infrastructure**: Switches, routers, protocols
- **Operational Data**: Charts, procedures, documentation

## ATA Chapters in L2-LINKS

- **ATA 23**: COMMUNICATIONS
  - VHF/HF radio systems
  - Satellite communications
  - Data link systems (ACARS, CPDLC)
  - Intercom and PA systems
  
- **ATA 42**: INTEGRATED MODULAR AVIONICS - NETWORK FABRIC
  - AFDX (ARINC 664) network
  - Ethernet switches
  - Network protocols and routing
  - Bandwidth management
  
- **ATA 91**: CHARTS / FLIGHT OPERATIONS
  - Electronic flight bag (EFB) data
  - Navigation charts
  - Operational procedures
  - Flight planning data

## AMPEL360 Specific Considerations

The hydrogen-electric BWB configuration requires:
- High-bandwidth network for distributed propulsion
- Redundant communication paths for safety
- Secure communications for hydrogen systems
- Real-time data links for system coordination
- Advanced connectivity for ground operations
