# 51-40: Load Cases and Design Criteria

## Overview
ATA 51-40 defines the design load cases, load factors, and structural design criteria for the AMPEL360 BWB aircraft, including maneuver, gust, ground, pressure, and thermal loads.

## Components
- **51-40-10:** Maneuver Load Cases
- **51-40-20:** Gust Load Cases
- **51-40-30:** Ground Load Cases
- **51-40-40:** Pressure Load Cases
- **51-40-50:** Thermal Load Cases
- **51-40-60:** Load Combination Rules

## Primary Load Cases
**Maneuver:** +2.5g positive, -1.0g negative (at MTOW 120,000 kg)
**Gust:** 50 ft/sec at Vc (cruise speed, Mach 0.85)
**Ground:** 2.0g landing, 1.0g braking (at MLW 105,000 kg)
**Pressure:** 8.5 psi differential at 41,000 ft
**Thermal:** -100°C cold-soak near LH₂ tank

## Design Philosophy
- Ultimate Load: 1.5× limit load (CS-25.303)
- Margin of Safety: >15% at ultimate load
- Critical Case: +2.5g pull-up + max cabin pressure
