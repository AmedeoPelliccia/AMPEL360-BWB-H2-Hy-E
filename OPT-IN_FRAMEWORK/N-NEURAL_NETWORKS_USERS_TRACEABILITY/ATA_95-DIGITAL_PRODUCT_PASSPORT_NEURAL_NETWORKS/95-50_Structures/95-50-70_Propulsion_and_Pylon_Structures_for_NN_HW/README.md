## Overview

This directory contains hardware structures and physical integration points for **Propulsion and Pylon Structures** components within the Neural Network (NN) hardware infrastructure.

Pylon equipment brackets, nacelle-mounted sensor assemblies, thermal and vibration isolation structures for propulsion system sensors and controls.
