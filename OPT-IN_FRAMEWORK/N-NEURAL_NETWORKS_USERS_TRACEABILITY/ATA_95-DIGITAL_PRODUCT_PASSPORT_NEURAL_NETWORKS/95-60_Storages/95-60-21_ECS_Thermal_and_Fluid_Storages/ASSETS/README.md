**Version:** 1.0

## Purpose

This directory contains all design artifacts, diagrams, data files, and configuration templates for storage systems in ATA 21, following the AMPEL360 ASSETS Standard.

---

## File Naming Convention

All assets follow the pattern:
```
95-60-21-A<nnn>_<CATEGORY>_<ShortName>.<ext>
```

Where:
- **A<nnn>**: Asset ID (e.g., A001, A101)
- **CATEGORY**: Asset type (DIAG, DRWG, MODL, DATA, etc.)
- **ShortName**: Descriptive name

---

## Asset Categories

- **A101-A199**: DIAGRAMS - Architecture, flow, and context diagrams
- **A201-A299**: DRAWINGS - Engineering drawings and prints
- **A301-A399**: MODELS - 3D models and schemas
- **A801-A899**: DATA - Configuration data and tables
- **A901-A999**: EXPORTS - Rendered outputs (PDF, PNG)
