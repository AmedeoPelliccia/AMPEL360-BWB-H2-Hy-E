# Model Engineering ASSETS

## Purpose

This folder contains diagrams, templates, and reference materials supporting model engineering activities.

## Contents

### Diagrams

- **95-00-06-01-A-001_Model_Architecture_Taxonomy.drawio** - Draw.io source for architecture taxonomy
- **95-00-06-01-A-002_Model_Architecture_Taxonomy.svg** - Visual taxonomy of all model architectures
- **95-00-06-01-A-003_Model_Versioning_Flow.svg** - Model versioning and release workflow

### Templates

Located in `Templates/` subdirectory:

- **95-00-06-01-A-101_Model_Spec_Template.md** - Complete model specification template
- **95-00-06-01-A-102_Loss_Spec_Template.md** - Loss function specification template

## Usage

### Model Specification Template

Use the model specification template when:
- Creating a new neural network model
- Documenting an existing model for deployment
- Preparing certification documentation

The template includes:
- Model architecture details
- Input/output specifications
- Training configuration
- Performance metrics
- Safety considerations
- Requirements traceability

### Loss Function Template

Use the loss function template when:
- Defining a custom loss function
- Documenting domain-specific objectives
- Specifying multi-objective optimization

## Diagram Formats

- **`.drawio`**: Editable source files (open with draw.io or diagrams.net)
- **`.svg`**: Scalable vector graphics for documentation (embed in Markdown)
- **`.png`**: Raster graphics for presentations (to be added)

## Document Naming Convention

Format: `95-00-06-01-A-{Number}_{Title}.{ext}`

- `A` indicates ASSET
- Numbers 001-099: Diagrams and visualizations
- Numbers 100+: Templates and reference materials

## Maintenance

- Update diagrams when architecture taxonomy changes
- Keep templates synchronized with latest standards
