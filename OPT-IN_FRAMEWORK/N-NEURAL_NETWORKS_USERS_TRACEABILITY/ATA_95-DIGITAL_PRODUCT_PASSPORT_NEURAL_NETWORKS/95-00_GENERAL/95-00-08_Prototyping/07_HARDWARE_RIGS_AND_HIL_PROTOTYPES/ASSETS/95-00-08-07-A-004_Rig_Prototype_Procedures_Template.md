# Rig Prototype Procedures

## Setup Procedure
1. Power on rig
2. Connect sensors
3. Verify IO mapping
4. Run self-test
5. Calibrate sensors

## Test Execution
1. Load test scenario
2. Start data acquisition
3. Run prototype model
4. Monitor outputs
5. Record results

## Shutdown Procedure
1. Stop data acquisition
2. Save results
3. Disconnect sensors
4. Power off rig
