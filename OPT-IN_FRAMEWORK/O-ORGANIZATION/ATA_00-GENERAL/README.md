# ATA 00 — GENERAL

## Overview

This is ATA Chapter 00: GENERAL, part of the O — ORGANIZATION axis.

## Structure

This chapter follows the mandatory OPT-IN Framework structure:

### 00-00_GENERAL (Lifecycle Folders)

The GENERAL layer contains 14 mandatory lifecycle folders covering the complete development cycle:

1. **00-00-01_Overview**: System overview and global architecture
2. **00-00-02_Safety**: Safety framework and analysis
3. **00-00-03_Requirements**: Requirements and traceability
4. **00-00-04_Design**: Design specifications and patterns
5. **00-00-05_Interfaces**: Interface control documents
6. **00-00-06_Engineering**: Analysis, models, and simulation
7. **00-00-07_V_AND_V**: Verification and validation
8. **00-00-08_Prototyping**: Prototype development
9. **00-00-09_Production_Planning**: Manufacturing planning
10. **00-00-10_Certification**: Certification evidence
11. **00-00-11_EIS_Versions_Tags**: Configuration management
12. **00-00-12_Services**: Maintenance and service
13. **00-00-13_Subsystems_Components**: Component breakdown
14. **00-00-14_Ops_Std_Sustain**: Operational standards

### Cross-ATA Root Buckets

The following buckets are mandatory in every ATA chapter:

- **00-10_Operations**: Operational procedures and use cases
- **00-20_Subsystems**: Functional subsystems (design-driven internal structure)
- **00-30_Circularity**: Sustainability, LCA, and circular economy
- **00-40_Software**: Software, control logic, and AI/ML
- **00-50_Structures**: Physical structures and frames
- **00-60_Storages**: Tanks, reservoirs, and storage
- **00-70_Propulsion**: Propulsion interfaces (if applicable)
- **00-80_Energy**: Energy management and distribution
- **00-90_Tables_Schemas_Diagrams**: Data tables and documentation
