# ATA 85 — INFRASTRUCTURE INTERFACE STANDARDS

## Overview

This is ATA Chapter 85: INFRASTRUCTURE INTERFACE STANDARDS, part of the I — INFRASTRUCTURES axis.

## Structure

This chapter follows the mandatory OPT-IN Framework structure:

### 85-00_GENERAL (Lifecycle Folders)

The GENERAL layer contains 14 mandatory lifecycle folders covering the complete development cycle:

1. **85-00-01_Overview**: System overview and global architecture
2. **85-00-02_Safety**: Safety framework and analysis
3. **85-00-03_Requirements**: Requirements and traceability
4. **85-00-04_Design**: Design specifications and patterns
5. **85-00-05_Interfaces**: Interface control documents
6. **85-00-06_Engineering**: Analysis, models, and simulation
7. **85-00-07_V_AND_V**: Verification and validation
8. **85-00-08_Prototyping**: Prototype development
9. **85-00-09_Production_Planning**: Manufacturing planning
10. **85-00-10_Certification**: Certification evidence
11. **85-00-11_EIS_Versions_Tags**: Configuration management
12. **85-00-12_Services**: Maintenance and service
13. **85-00-13_Subsystems_Components**: Component breakdown
14. **85-00-14_Ops_Std_Sustain**: Operational standards

### Cross-ATA Root Buckets

The following buckets are mandatory in every ATA chapter:

- **85-10_Operations**: Operational procedures and use cases
- **85-20_Subsystems**: Functional subsystems (design-driven internal structure)
- **85-30_Circularity**: Sustainability, LCA, and circular economy
- **85-40_Software**: Software, control logic, and AI/ML
- **85-50_Structures**: Physical structures and frames
- **85-60_Storages**: Tanks, reservoirs, and storage
- **85-70_Propulsion**: Propulsion interfaces (if applicable)
- **85-80_Energy**: Energy management and distribution
- **85-90_Tables_Schemas_Diagrams**: Data tables and documentation
