# 85-00-03-EMERG-001 — Fire and Rescue Infrastructure Interface Requirements

## 1. Requirement Statement

Airport Rescue and Firefighting (ARFF) infrastructure **SHALL** support:
- **ARFF Category**: Minimum Category 8 per [ICAO Annex 14](https://www.icao.int/safety/pages/annex-14.aspx), Table 9-1
- **Response Time**: ARFF vehicles reach any point on runway within 3 minutes
- **Access**: All-weather access routes to all aircraft sides
- **Equipment**: Water/foam capacity per ICAO Category 8 requirements

## 2. Rationale

The aircraft's size and passenger capacity require Category 8 ARFF protection, ensuring rapid fire suppression and rescue capabilities in emergencies.

## 3. Category

- **Requirement Type**: Safety — Fire Protection
- **Domain**: Emergency Rescue and Safety

## 4. Acceptance Criteria

- [ ] Target airports classified as ARFF Category 8 or higher
- [ ] ARFF response time ≤ 3 minutes validated
- [ ] Access routes to all aircraft sides confirmed
- [ ] ARFF equipment inventory meets ICAO Category 8 requirements
