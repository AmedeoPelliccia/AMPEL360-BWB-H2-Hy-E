# 85-00-03-CO2BAT — CO₂ Battery and Hybrid Storage Interface

## Purpose

This folder contains requirements for the **CO₂ Battery and Hybrid Storage Interface** of the AMPEL360 BWB H₂ Hy-E aircraft. These requirements define ground-based charging and servicing interfaces for the aircraft's innovative CO₂ capture battery system and hybrid energy storage.

## Scope

- **CO₂ Battery Charging**: Electrical and thermal management interfaces
- **Hybrid Storage Operations**: Charging protocols for hybrid energy systems
- **CO₂ Off-loading**: Transfer systems for captured CO₂ from aircraft to ground storage
- **Safety Protocols**: CO₂ battery incident procedures, thermal runaway mitigation
