# DPP Integration Links

This directory contains configuration files that map service events and activities to Digital Product Passport (DPP) entries for ATA 85 infrastructure components.

## Contents

- **85-00-12-A-401_Service_Events_to_DPP_Map.csv**: Mapping between service event types (incidents, maintenance, changes) and corresponding DPP update categories
- **85-00-12-A-402_Service_Configuration_Anchors.json**: Configuration anchors that link service management system records to DPP component identifiers

## Purpose

These files enable automatic integration between service operations and the DPP system, ensuring:
- Complete lifecycle traceability for infrastructure components
- Automatic DPP updates when service activities occur
- Integration with digital twin for real-time status
- Compliance with circularity and sustainability tracking requirements

## Usage

These configuration files are consumed by the service management system (ITSM) and DPP integration middleware.
