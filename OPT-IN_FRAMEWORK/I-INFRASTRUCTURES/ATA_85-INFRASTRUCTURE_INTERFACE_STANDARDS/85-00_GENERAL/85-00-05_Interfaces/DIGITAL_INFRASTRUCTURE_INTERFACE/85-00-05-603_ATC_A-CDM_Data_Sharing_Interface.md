# 85-00-05-603: ATC A-CDM Data Sharing Interface

## 1. Introduction
This document defines the interface for sharing data with Air Traffic Control (ATC) as part of Airport Collaborative Decision Making ([A-CDM](https://www.eurocontrol.int/concept/airport-collaborative-decision-making)).

## 2. Data Elements
*   **TSAT:** Target Start-up Approval Time.
*   **TTOT:** Target Take-off Time.
*   **De-icing:** Request and status.

## 3. Requirements
*   **REQ-D-201:** The FMS shall be capable of receiving and displaying TSAT messages.
*   **REQ-D-202:** Updates to TOBT shall be automatically pushed to the [A-CDM](https://www.eurocontrol.int/concept/airport-collaborative-decision-making) platform.
