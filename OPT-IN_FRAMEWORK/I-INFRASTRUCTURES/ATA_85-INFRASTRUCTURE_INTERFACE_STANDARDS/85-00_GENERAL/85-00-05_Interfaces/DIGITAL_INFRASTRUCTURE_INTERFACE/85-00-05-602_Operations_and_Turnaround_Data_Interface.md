# 85-00-05-602: Operations and Turnaround Data Interface

## 1. Introduction
This document defines the data interface for monitoring turnaround milestones (Target Off-Block Time - TOBT).

## 2. Milestones
*   **ACARS Events:** OOOI (Out, Off, On, In).
*   **Service Events:** Refuelling Start/Stop, Cleaning Complete, Catering Complete.

## 3. Requirements
*   **REQ-D-101:** The aircraft shall broadcast service status updates to the ground handler's system.
*   **REQ-D-102:** H2 refuelling status (Pressure, Temp) shall be monitored in real-time.
