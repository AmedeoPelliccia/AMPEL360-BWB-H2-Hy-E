# 85-00-05-102: Apron Stand Positioning Interface ICD

## 1. Introduction
This document defines the interface for positioning the BWB aircraft on an airport stand, including stop bar locations and alignment guidance.

## 2. Interface Description

### 2.1. Visual Docking Guidance System (VDGS)
*   **Type:** Laser/Camera based (e.g., Safedock).
*   **Configuration:** Specific software profile required for BWB geometry.
*   **Stop Position:** Defined relative to the nose gear.

### 2.2. Marshalling
*   **Hand Signals:** Standard [IATA ground operations hand signals](https://www.iata.org/en/programs/ops-infra/ground-operations/safety/) apply.
*   **Blind Spots:** Pilot visibility diagram provided in [ASSETS](./ASSETS/85-00-05-A-102_Stand_Clearance_Envelope.svg).

### 2.3. Towing
*   **Towbar:** Compatible with [ATA 09] towbar standards.
*   **Towbarless:** Nose gear geometry compatible with standard TBL tractors.

## 3. Requirements
*   **REQ-A-101:** The aircraft shall be detectable by standard VDGS units.
*   **REQ-A-102:** Stop positions shall ensure PBB connection without exceeding extension limits.
