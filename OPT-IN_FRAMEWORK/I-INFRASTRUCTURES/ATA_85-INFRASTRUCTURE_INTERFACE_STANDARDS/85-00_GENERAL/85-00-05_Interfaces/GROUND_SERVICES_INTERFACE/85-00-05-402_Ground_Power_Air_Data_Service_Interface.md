# 85-00-05-402: Ground Power Air Data Service Interface

## 1. Introduction
This document defines the interfaces for Ground Power Units (GPU) and Pre-Conditioned Air (PCA).

## 2. Ground Power
*   **Standard:** [ISO 6858](https://www.iso.org/obp/ui/#iso:std:iso:6858:ed-2:v1:en).
*   **Voltage:** 115V AC 400Hz / 270V DC.
*   **Connector:** Standard 6-pin (AC) or appropriate DC connector.
*   **Location:** Nose section / Forward fuselage.

## 3. Pre-Conditioned Air (PCA)
*   **Standard:** [IATA AHM](https://www.iata.org/en/publications/manuals/airport-handling-manual/) 973.
*   **Connector:** 8-inch standard coupling.
*   **Location:** Underbelly / Wing root.

## 4. Requirements
*   **REQ-G-101:** GPU connection shall support "Power-by-Wire" interlock.
*   **REQ-G-102:** PCA hose routing shall not obstruct emergency egress paths.
