# ATA 85-00-05: Interfaces

## 1. Overview
This directory contains the comprehensive interface documentation for the AMPEL360 BWB H2-Hybrid Aircraft infrastructure standards. It defines the physical, digital, and operational boundaries between the aircraft and the airport environment.

## 2. Structure
The interface documentation is organized into the following categories:

*   **[MASTER](./MASTER/README.md)**: Central catalog and cross-reference maps.
*   **[AIRPORT_INTERFACE](./AIRPORT_INTERFACE/README.md)**: Geometric and physical airport integration.
*   **[H2_INFRASTRUCTURE_INTERFACE](./H2_INFRASTRUCTURE_INTERFACE/README.md)**: Hydrogen refuelling and safety interfaces.
*   **[CO2_BATTERY_INTERFACE](./CO2_BATTERY_INTERFACE/README.md)**: CO2 battery handling and service interfaces.
*   **[GROUND_SERVICES_INTERFACE](./GROUND_SERVICES_INTERFACE/README.md)**: General ground support equipment (GSE) interfaces.
*   **[PAX_BOARDING_EVAC_INTERFACE](./PAX_BOARDING_EVAC_INTERFACE/README.md)**: Passenger boarding and emergency evacuation interfaces.
*   **[DIGITAL_INFRASTRUCTURE_INTERFACE](./DIGITAL_INFRASTRUCTURE_INTERFACE/README.md)**: Data exchange and IT integration.
*   **[ENERGY_INFRASTRUCTURE_INTERFACE](./ENERGY_INFRASTRUCTURE_INTERFACE/README.md)**: Electrical grid and power supply interfaces.
