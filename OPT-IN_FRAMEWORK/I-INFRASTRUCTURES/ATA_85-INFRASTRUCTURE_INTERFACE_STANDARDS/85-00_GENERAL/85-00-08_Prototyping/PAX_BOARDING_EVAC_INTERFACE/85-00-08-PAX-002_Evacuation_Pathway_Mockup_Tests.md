# 85-00-08-PAX-002 Evacuation Pathway Mockup Tests

## 1. Purpose

Define the evacuation pathway mockup test program for demonstrating compliance with [CS-25.803](https://www.easa.europa.eu/en/document-library/certification-specifications/cs-25-large-aeroplanes) 90-second evacuation requirements.

## 2. Scope

- Full-scale cabin mockup with representative exits
- Emergency evacuation demonstrations
- Slide deployment and functionality
- Evacuation pathway accessibility
- Crew emergency procedures validation

## 3. Key Performance Indicators

| KPI                                | Target              |
|------------------------------------|---------------------|
| **Evacuation Time**                | ≤ 90 sec            |
| **Exit Utilization**               | Balanced flow       |
| **Passenger Safety**               | Zero injuries       |

## 4. Regulatory Compliance

- [CS-25.803](https://www.easa.europa.eu/en/document-library/certification-specifications/cs-25-large-aeroplanes) – Emergency evacuation
- [CS-25.807](https://www.easa.europa.eu/en/document-library/certification-specifications/cs-25-large-aeroplanes) – Emergency exits
