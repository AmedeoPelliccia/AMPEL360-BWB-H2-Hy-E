# ATA 115 - Flight Simulator Systems

## Overview
This chapter contains the technical basis for the Full Flight Simulator (FFS) used for crew training and certification. It defines the systems architecture, data packages, and operational requirements.

## Structure

### 01-QUALIFICATION_AND_CERTIFICATION
Qualification standards and certification requirements for the FFS

### 02-AIRCRAFT_DATA_PACKAGE_OEM
OEM-provided aircraft data package for simulator development

### 03-SIMULATOR_SYSTEMS_ARCHITECTURE
Technical architecture of the simulator systems

### 04-OPERATIONS_AND_MAINTENANCE
Operating procedures and maintenance requirements for the simulator

### 05-DATABASES
Aerodynamic, engine, and systems databases

### 06-CONFIGURATION_AND_SOFTWARE
Configuration management and software documentation

### 07-REGISTRY_AND_LOGS
Registry of simulator configurations and operational logs

## Governance
The FFS must be qualified to the highest regulatory standards to ensure it accurately represents the AMPEL360 aircraft for training and certification purposes.
