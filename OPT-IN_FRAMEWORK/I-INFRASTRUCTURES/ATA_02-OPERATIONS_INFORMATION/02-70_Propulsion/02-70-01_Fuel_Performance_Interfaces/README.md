# 02-70-01 — Fuel Performance Interfaces

## Purpose

This subtree contains interfaces and data structures for H₂ fuel planning, consumption monitoring, range calculation, boiloff prediction, and fuel system health monitoring.

## Scope

- H₂ fuel planning integration with flight planning systems
- Real-time fuel consumption monitoring
- Range calculation based on current fuel state and performance
- Boiloff prediction models and data structures
- Fuel system health indicators
