# RQ-CLEARANCES

This directory contains ground clearance requirements for safe aircraft operations.

## Requirements

- RQ-02-11-30-001: Wingtip Clearance ≥1.2m Min
- RQ-02-11-30-002: Belly Clearance ≥1.8m Min
- RQ-02-11-30-003: Tail Clearance ≥2.5m Min
- RQ-02-11-30-004: Engine Intake Clearance 2.8m
- RQ-02-11-30-005: Ground Attitude Limits
- RQ-02-11-30-006: Pavement Requirements

## Category
Ground clearance specifications for safe taxi, takeoff, and landing operations.
