# RQ-02-10-00-002: Type Certificate Data

**Requirement ID:** RQ-02-10-00-002  
**Category:** General  
**Priority:** Critical  
**Status:** Approved

## Requirement

All aircraft shall be manufactured and maintained in accordance with EASA Type Certificate Data Sheet (TCDS) and associated documentation.

**Type Certificate Requirements:**
- Configuration controlled design data
- Approved flight manual
- Maintenance manual compliance
- Airworthiness limitations
- Equipment list
- Weight and balance data
- Performance data

## Rationale

Type Certificate Data ensures:
- Regulatory compliance
- Airworthiness standards adherence
- Configuration control
- Safety of flight operations
- Uniform manufacturing standards

## Verification

- **Method:** Inspection and Documentation Review
- **Procedure:** Verify conformity with TCDS requirements
- **Acceptance Criteria:**
  - All TCDS requirements documented
  - Configuration matches certified design
  - Deviations properly approved per Part 21

## Compliance

- EASA Part 21 Subpart C: Type Certificates
- CS-25: Certification Specifications
- TCDS approval by EASA
