# PNR-02-10-103 - CG Reference Markers

**Part Number:** PNR-02-10-103  
**Type:** Marking System  
**Manufacturer:** AMPEL360  
**Status:** Active

## Description

Center of Gravity (CG) reference marking system for weight and balance operations. Provides visual indicators for CG limits and datum reference points specific to the BWB configuration.

## Specifications

- **Material:** Anodized aluminum plates
- **Visibility:** High-contrast yellow/black
- **Mounting:** Permanent rivets
- **Location:** Fuselage floor, equipment bay
- **Marking Type:** Laser-engraved graduations

## Marker Types

### 1. Datum Reference Marker
- **Location:** Forward equipment bay floor
- **Marking:** "DATUM - Station 0.0"
- **Size:** 100mm × 50mm
- **Purpose:** Primary measurement reference point

### 2. CG Range Indicator
- **Location:** Cargo compartment floor
- **Markings:** 
  - Forward CG Limit: 15% MAC
  - Aft CG Limit: 35% MAC
  - Operating CG Range shaded area
- **Size:** 300mm × 150mm
- **Scale:** 1% MAC increments

### 3. Lateral CG Markers
- **Location:** Left and right cargo bay walls
- **Purpose:** Lateral balance verification
- **Range:** ±5% of span

## Usage

1. **Pre-flight Weight and Balance:**
   - Locate datum marker
   - Measure cargo positions relative to datum
   - Verify CG within marked limits

2. **Loading Operations:**
   - Use CG range indicator for cargo placement
   - Ensure balanced loading (lateral markers)
   - Document final CG position

3. **Maintenance:**
   - Reference for equipment installation
   - Verify structural modifications impact on CG

## CG Calculation Reference

```
CG Position = (Σ(Weight × Arm)) / Total Weight
MAC Reference: 0% at Station 450, 100% at Station 750
```
