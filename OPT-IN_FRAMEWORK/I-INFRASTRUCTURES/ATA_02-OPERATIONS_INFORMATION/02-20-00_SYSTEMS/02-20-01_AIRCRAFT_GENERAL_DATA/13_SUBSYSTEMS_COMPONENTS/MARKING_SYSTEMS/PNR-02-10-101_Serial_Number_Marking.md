# PNR-02-10-101 - Serial Number Marking

**Part Number:** PNR-02-10-101  
**Type:** Marking System  
**Manufacturer:** AMPEL360  
**Status:** Active

## Description

Permanent serial number marking system for primary aircraft structure identification. Provides traceable, tamper-evident identification compliant with aviation regulations.

## Specifications

- **Method:** Laser engraving / Chemical etching
- **Character Height:** 6mm minimum
- **Depth:** 0.1-0.2mm
- **Location:** Multiple critical structures
- **Durability:** Permanent, corrosion-resistant

## Marking Locations

1. **Primary Structure:**
   - Forward fuselage frame (Station 150)
   - Center wing box (Left & Right)
   - Aft fuselage bulkhead (Station 800)

2. **Major Assemblies:**
   - Main landing gear attachment points
   - Engine mount structures
   - Hydrogen tank mounting frames

## Format

```
AMPEL360-BWB-H2-[SERIAL]
Manufacturing Date: [MM/YYYY]
```

## Application Process

1. Surface preparation (cleaning, degreasing)
2. Template positioning and verification
3. Laser engraving or chemical etching
4. Quality inspection
5. Protective coating application
