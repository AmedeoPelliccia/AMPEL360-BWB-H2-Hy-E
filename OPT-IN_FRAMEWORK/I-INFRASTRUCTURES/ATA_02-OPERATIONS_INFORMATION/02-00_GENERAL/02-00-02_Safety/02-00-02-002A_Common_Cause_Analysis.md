# Common Cause Analysis (CCA)

- Physical CCCs: shared power/comms, shared data sources (airport databases), co-located infra.
- Functional CCCs: shared procedures, shared roles, common publication pipeline.
- Latent CCCs: configuration errors, stale data ingestion, publication delays.

> Interfaces with: `FHA`, `FMEA`, and `Runtime Monitoring`.
