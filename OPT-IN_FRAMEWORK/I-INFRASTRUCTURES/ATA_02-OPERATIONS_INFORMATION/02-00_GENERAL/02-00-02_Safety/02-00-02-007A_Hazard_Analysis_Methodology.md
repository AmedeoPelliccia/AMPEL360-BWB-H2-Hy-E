# Hazard Analysis Methodology

- Process: Identification → Classification → Mitigation → Verification → Residual risk.
- Sources: Ops processes, datasets, airport interfaces, tooling.
- Classifications: Catastrophic/Hazardous/Major/Minor/No Effect (tie to DAL A–E).
- Traceability: Each hazard links to **requirements**, **tests**, **evidence**.
