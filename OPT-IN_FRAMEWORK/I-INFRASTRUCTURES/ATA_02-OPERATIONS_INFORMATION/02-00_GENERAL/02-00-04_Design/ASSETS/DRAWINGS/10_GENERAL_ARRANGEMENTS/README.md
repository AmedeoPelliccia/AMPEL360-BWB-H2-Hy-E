## Purpose

This directory contains **general arrangement (GA) drawings** showing overall layouts and spatial organization:

- Operations centers, control rooms, and dispatch areas
- Data centers and server rooms
- H2 storage and handling facilities
- GSE areas and support spaces
- Overall site layouts and building interfaces

---

## Drawing Domain

All drawings in this directory use domain code: **GA** (General Arrangements)

---

## Typical Content

GA drawings typically include:

- Floor plans showing room layouts and major equipment locations
- Spatial relationships between functional areas
- Circulation paths and egress routes
- Key dimensions and clearances
- Door locations and swing directions
- Callouts to interface, detail, and equipment drawings
- Furniture and console layouts for operational spaces

---

## Naming Convention

```text
02-00-04-DWG-GA-[Location]-[Description]-[SheetID]-R[rev].[ext]
```

**Examples:**

- `02-00-04-DWG-GA-OPSCTR-OverallLayout-SHT01-R01.dwg`
- `02-00-04-DWG-GA-DATACENTER-OverallLayout-SHT01-R01.dwg`
- `02-00-04-DWG-GA-H2ROOM-GeneralArrangement-SHT01-R01.dwg`

## Sidecar Descriptions

Each `.dwg` file should have a corresponding `.md` file with the same basename providing:

- Drawing identification and metadata
- Purpose and scope
- Source BIM models and references
- Content summary
- Assumptions and constraints
- Revision history
- Approval information

See `DRAWING_DESCRIPTION_TEMPLATE.md` in `00_ADMIN/` for the standard template.
