## Purpose

This directory contains **equipment and GSE (Ground Support Equipment) assembly drawings**:

- Equipment general assemblies showing overall configuration
- GSE power carts, tool carts, and support equipment
- Server racks and IT equipment assemblies
- Console assemblies for operations centers
- Specialized equipment for hydrogen handling and storage

---

## Drawing Domain

All drawings in this directory use domain code: **EQP** (Equipment / GSE drawings)

---

## Typical Content

Equipment drawings typically include:

- Isometric and orthogonal views of equipment assemblies
- Overall dimensions and weight specifications
- Major component callouts and bill of materials references
- Connection points (power, data, fluid, mechanical)
- Mounting and anchoring details
- Mobility features (wheels, casters, lifting points)
- Operating clearances and maintenance access requirements
- Assembly notes and critical tolerances

---

## Naming Convention

```text
02-00-04-DWG-EQP-[Equipment]-[Description]-[SheetID]-R[rev].[ext]
```

**Examples:**

- `02-00-04-DWG-EQP-GSE_PowerCart-GeneralAssembly-SHT01-R01.dwg`
- `02-00-04-DWG-EQP-ServerRack-Assembly-SHT01-R01.dwg`
- `02-00-04-DWG-EQP-OperatorConsole-Assembly-SHT01-R01.dwg`

## Sidecar Descriptions

Each `.dwg` file should have a corresponding `.md` file with the same basename providing:

- Drawing identification and metadata
- Purpose and scope
- Source BIM/CAD models and assembly references
- Content summary
- Assumptions and constraints
- Revision history
- Approval information

See `DRAWING_DESCRIPTION_TEMPLATE.md` in `00_ADMIN/` for the standard template.
