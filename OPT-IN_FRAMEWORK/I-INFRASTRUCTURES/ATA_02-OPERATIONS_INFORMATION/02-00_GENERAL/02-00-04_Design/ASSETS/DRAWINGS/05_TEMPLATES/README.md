# 05_TEMPLATES — Drawing Templates

## Purpose

This directory contains **standard drawing templates** used across all DRAWINGS subdirectories:

- Title blocks for various sheet sizes (A3, A1, etc.)
- Standard drawing frames with borders and zones
- Layer standards and line type definitions
- Symbol libraries and blocks
- Text styles and dimension styles

---

## Contents

Templates follow the naming pattern:

```text
02-00-04-TPL-DWG-[TemplateType]-[SheetSize]-R[rev].[ext]
```

**Example files:**

- `02-00-04-TPL-DWG-TitleBlock-A3-R01.dwg` — A3 sheet title block template
- `02-00-04-TPL-DWG-TitleBlock-A1-R01.dwg` — A1 sheet title block template
- `02-00-04-TPL-DWG-DrawingFrame-R01.dwg` — Standard drawing frame/border

---

## Usage

1. Open the appropriate template for your drawing type and sheet size
2. Save as a new drawing using the standard naming convention
3. Fill in title block information (drawing ID, title, revision, etc.)
4. Create drawing content following layer and style standards
5. Do not modify the template files directly; propose changes through design review

---

## Template Management

- Templates are version-controlled with revision numbers (R01, R02, etc.)
- All template changes must be approved by the design lead
- When a template is revised, existing drawings are not automatically updated
- Consider creating a dated archive in `99_ARCHIVE/` when releasing new template versions
