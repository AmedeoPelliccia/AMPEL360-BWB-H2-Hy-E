# ASSETS Directory

This directory contains visual assets supporting the circularity documentation:

- **Diagrams**: Architecture diagrams, flow charts, and visual frameworks (SVG, DrawIO)
- **Spreadsheets**: Templates and calculation worksheets (XLSX)
- **Images**: Mockups, screenshots, and illustrations (PNG, JPG)
- **Model Cards**: ML model documentation with carbon footprint data (YAML)

## Naming Convention

Assets follow the pattern: `{subsystem-id}-A-{sequence}_Description.{ext}`

Example: `02-30-02-A-001_Hardware_Lifecycle_Flow.svg`

## Status

Asset files are placeholders pending development of detailed technical diagrams.
