### Components Displayed
1. **Tank Status (4 tanks)**
   - Quantity (kg)
   - Temperature (°C)
   - Pressure (bar)
   - Status indicator

2. **Flow Lines**
   - Animated flow indicators
   - Flow rates (kg/hr)
   - Valve positions

3. **Fuel Cell Connections**
   - FC1-2 (left system)
   - FC3-4 (right system)
   - Power output per pair

4. **System Status Panel**
   - Total quantity
   - Total flow rate
   - Leak detection status
   - Ventilation status

### Color Coding
- Green: Normal operation (all parameters in range)
- Amber: Caution (minor leak, temp/pressure caution)
- Red: Warning (major leak, emergency condition)
- Blue: Valve closed/inactive
- White: Flow lines (animated when active)

### Interactive Elements
- Clickable components for detailed info
- Valve status indicators
- Trend displays for parameters
- Historical data access

### Alert Integration
- Visual alerts on affected components
- Master warning/caution integration
- Aural alert coordination
- Procedure link (one-touch access to emergency procedures)
