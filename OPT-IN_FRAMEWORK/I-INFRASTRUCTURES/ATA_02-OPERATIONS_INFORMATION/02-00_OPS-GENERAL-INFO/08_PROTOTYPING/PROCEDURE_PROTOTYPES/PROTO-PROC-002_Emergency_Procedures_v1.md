## Overview

Emergency procedures for critical system failures in the AMPEL360 BWB H2 Hy-E aircraft.

## H2 Leak Detection

### Memory Items (Immediate Action)
1. H2 MAIN VALVE - CLOSE
2. FUEL CELL SHUTDOWN - EXECUTE
3. BATTERY MODE - ACTIVATE
4. VENTILATION - MAXIMUM

### Subsequent Actions
1. H2 leak location - IDENTIFY
2. CAOS diagnostic - REVIEW
3. Nearest suitable airport - SELECT
4. Emergency services - NOTIFY
5. QRH procedure - FOLLOW

## Fuel Cell Failure

### Single Stack Failure
1. Failed stack - ISOLATE
2. Remaining stack - INCREASE OUTPUT
3. Battery assist - ENGAGE
4. Flight plan - REVISE
5. Land at nearest suitable airport

### Dual Stack Failure
1. Both stacks - SHUTDOWN
2. Battery emergency mode - ACTIVATE
3. Non-essential systems - SHUTDOWN
4. Emergency descent - INITIATE
5. Declare MAYDAY

## Battery Critical

1. Load shedding - EXECUTE
2. Essential systems only
3. H2 fuel cell - MAXIMUM OUTPUT
4. Land ASAP

## Loss of CAOS System

1. Revert to manual procedures
2. Traditional instruments - USE
3. QRH paper backup - REFERENCE
4. Continue flight with caution
