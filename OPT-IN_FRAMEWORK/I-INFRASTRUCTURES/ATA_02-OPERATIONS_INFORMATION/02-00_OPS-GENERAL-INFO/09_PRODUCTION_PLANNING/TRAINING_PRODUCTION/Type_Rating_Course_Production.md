# Type Rating Course Production Plan

**Course Code:** AMPEL360-TR-001  
**Duration:** 42 days (Ground School + Simulator + Line Training)  
**Production Period:** 2025-12-15 to 2026-05-15  
**Status:** Active  
**Owner:** Training

## Overview
Development plan for the EASA/FAA Type Rating Course for the AMPEL360 BWB H₂ Hy-E aircraft. Comprehensive training program for pilots transitioning to this revolutionary aircraft.

## Course Structure
### Phase 1: Ground School (14 days)
#### Week 1: Aircraft Systems
- BWB aerodynamics and handling
- H₂ fuel system operations
- Fuel cell propulsion system
- Electrical power generation
- Flight control systems
- CAOS system introduction

#### Week 2: Advanced Systems & Performance
- CO₂ capture system
- Environmental control systems
- Performance characteristics
- Weight and balance (BWB specific)
- Fuel planning (H₂ considerations)
- Emergency equipment

### Phase 2: Simulator Training (21 days)
#### Normal Operations (7 days)
- Cockpit familiarization
- Pre-flight procedures
- Engine start (fuel cell activation)
- Taxi, takeoff, landing
- Standard procedures
- CAOS system operation

#### Non-Normal & Emergency (7 days)
- System failures
- H₂ system emergencies
- Engine (fuel cell) failures
- Electrical failures
- CAOS degradation procedures
- Emergency landings

#### Line-Oriented Flight Training - LOFT (7 days)
- Complete flight scenarios
- Weather challenges
- System integration
- Crew resource management
- Decision making
- CAOS-assisted operations

### Phase 3: Line Training (7 days minimum)
- Actual aircraft operations
- Route familiarization
- Line operations under supervision
- H₂ refueling procedures
- Normal and abnormal situations
- Final competency check

## Learning Objectives
### Knowledge Objectives
- Understand BWB aerodynamic principles
- Master H₂ fuel system operations
- Comprehend fuel cell power generation
- Operate CAOS system effectively
- Apply emergency procedures correctly
- Calculate performance accurately

### Skill Objectives
- Fly BWB aircraft safely
- Manage H₂ fuel system
- Operate all aircraft systems
- Use CAOS advisory system
- Handle emergencies competently
- Work effectively as crew
