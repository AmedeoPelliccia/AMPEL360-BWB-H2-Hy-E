# EFB Loading Procedures

**Scope:** All operational documentation on EFB  
**Owner:** IT / Flight_Ops

## EFB Content
- Aircraft Flight Manual (AFM)
- Flight Crew Operating Manual (FCOM)
- Quick Reference Handbook (QRH)
- Weight and Balance Manual (WBM)
- Minimum Equipment List (MEL)
- Charts and navigation data
- CAOS interface

## Loading Process
1. Content validation and approval
2. Format conversion for EFB
3. Package creation
4. Quality testing
5. Upload to EFB management system
6. Aircraft EFB synchronization
7. Crew notification

## Update Procedures
- Critical updates: Immediate push
- Important updates: Within 24 hours
- Routine updates: Weekly cycle
- Scheduled updates: Monthly cycle
