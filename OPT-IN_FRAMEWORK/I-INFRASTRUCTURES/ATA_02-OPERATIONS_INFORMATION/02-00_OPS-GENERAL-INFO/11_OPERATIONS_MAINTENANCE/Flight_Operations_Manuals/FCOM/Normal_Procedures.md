# Normal Procedures

## Pre-Flight
□ CAOS System - CHECK  
□ H₂ Fuel Quantity - VERIFY  
□ Fuel Cell Status - CHECK  
□ Digital Twin - SYNCHRONIZED  
□ Weather Data - LOADED  
□ Flight Plan - REVIEWED

## Before Start
□ Parking Brake - SET  
□ H₂ Master - OFF  
□ All Fuel Cells - OFF  
□ APU - AS REQUIRED

## Fuel Cell Start
□ H₂ Master - ON  
□ Monitor H₂ pressure (>300 bar)  
□ Fuel Cell 1 - ON (wait for stabilization)  
□ Fuel Cell 2 - ON  
□ Fuel Cell 3 - ON  
□ Fuel Cell 4 - ON  
□ All parameters - CHECK GREEN

## Taxi
□ CAOS Taxi Mode - ENGAGED  
□ H₂ System - MONITOR  
□ Brakes - TEST

## Before Takeoff
□ Flight Controls - CHECK  
□ CAOS Takeoff Data - CONFIRM  
□ H₂ Fuel - SUFFICIENT  
□ All Systems - NORMAL

## After Landing
□ Fuel Cells - REDUCE LOAD  
□ CAOS Post-Flight - REVIEW  
□ H₂ System - MONITOR

## Shutdown
□ Fuel Cells - OFF (sequential)  
□ H₂ Master - OFF  
□ CAOS - SAVE FLIGHT DATA
