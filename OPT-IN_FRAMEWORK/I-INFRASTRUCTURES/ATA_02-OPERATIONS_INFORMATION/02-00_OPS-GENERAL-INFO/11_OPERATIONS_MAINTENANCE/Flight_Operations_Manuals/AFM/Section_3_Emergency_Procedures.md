# Section 3 - Emergency Procedures

## 3.1 H₂ Fuel Leak
**CONDITION:** H₂ LEAK DETECTED alert

**MEMORY ITEMS:**
1. H₂ Master Switch - OFF
2. Fuel Cell Affected - OFF
3. Emergency Power - BATTERY
4. Land as soon as possible

## 3.2 Fuel Cell Failure
**CONDITION:** FUEL CELL FAIL alert

**MEMORY ITEMS:**
1. Failed Fuel Cell - OFF
2. Load Shed per ECAM
3. Assess remaining power capability
4. Land at nearest suitable airport

## 3.3 Complete Power Failure
**CONDITION:** All fuel cells offline

**MEMORY ITEMS:**
1. Battery Power - ON
2. Essential Bus only
3. Declare emergency
4. Immediate landing required

## 3.4 CAOS System Failure
**CONDITION:** CAOS FAIL alert

**ACTION:**
- Revert to manual operations
- Aircraft remains fully operational
- Follow standard procedures
