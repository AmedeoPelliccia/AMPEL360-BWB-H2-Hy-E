# Proprietary Marking Guide

**Document ID:** AMPEL360-02-00-00-META-PROP  
**Version:** 1.0.0  
**Effective Date:** 2025-09-01

## Purpose

This guide defines the proprietary marking requirements for AMPEL360 documentation to protect intellectual property and comply with export control regulations.

## Classification Levels

### PUBLIC
- **Mark**: None required
- **Access**: Unrestricted
- **Distribution**: Open
- **Examples**: Press releases, marketing materials

### INTERNAL
- **Mark**: "AMPEL360 INTERNAL USE ONLY"
- **Header/Footer**: On every page
- **Access**: AMPEL360 employees only
- **Distribution**: Internal distribution lists
- **Examples**: Process documents, internal procedures

### CONFIDENTIAL
- **Mark**: "AMPEL360 CONFIDENTIAL"
- **Header/Footer**: On every page
- **Watermark**: Optional diagonal watermark
- **Access**: Need-to-know basis
- **Distribution**: Controlled distribution, logged
- **Examples**: Design details, technical specifications

### RESTRICTED
- **Mark**: "AMPEL360 RESTRICTED - AUTHORIZED PERSONNEL ONLY"
- **Header/Footer**: On every page  
- **Watermark**: Mandatory red diagonal watermark
- **Access**: Specific approval required
- **Distribution**: Highly limited, individually authorized
- **Examples**: Proprietary technologies, trade secrets

### EXPORT CONTROLLED
- **Mark**: "EXPORT CONTROLLED - SEE EXPORT CONTROL COMPLIANCE"
- **Header/Footer**: On every page
- **Additional**: ITAR/EAR notice
- **Access**: Export license may be required
- **Distribution**: Strict export control compliance
- **Examples**: Advanced technologies subject to export regulations

### Cover Page
Must include:
- Classification level (prominent)
- Distribution statement
- Proprietary notice
- Copyright notice
- Export control notice (if applicable)

### Electronic Documents
- Metadata classification tag
- PDF security settings appropriate to classification
- Watermarking (if required)
- Access controls enforced

## Distribution Statements

### Statement A (PUBLIC)
> "Approved for public release; distribution unlimited."

### Statement B (INTERNAL)
> "Distribution authorized to AMPEL360 personnel only."

### Statement C (CONFIDENTIAL)
> "Distribution authorized to AMPEL360 and authorized partners only; contains proprietary information."

### Statement D (RESTRICTED)
> "Distribution authorized to specifically approved recipients only; contains trade secrets and proprietary information."

### Statement E (EXPORT CONTROLLED)
> "Distribution subject to export control regulations. Export of this document or technical data contained herein may require an export license."

## Handling Requirements

### CONFIDENTIAL and Above

**Storage:**
- Locked cabinets or secure rooms
- Encrypted electronic storage
- Access logged

**Transmission:**
- Encrypted email
- Secure file transfer
- Hand delivery with receipt

**Copying:**
- Approval required
- Copy numbered and logged
- Original markings retained

**Disposal:**
- Shredding (paper)
- Secure data wiping (electronic)
- Disposal logged

## Export Control Compliance

Documents containing export-controlled technical data must:
1. Be marked with export control notice
2. Have export control classification determined
3. Be handled per Export_Control_Compliance.md
4. Have distribution approval from Export Control Officer
