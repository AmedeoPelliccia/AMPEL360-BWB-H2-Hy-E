# Reference Coordinate Systems

**Document ID:** AMPEL360-02-11-00-OVW-003  
**Version:** 1.0.0

## Body Axis System

**Origin:** Nose apex (forward-most point)

**Axes Definition:**
- **X-axis:** Longitudinal, positive forward (nose direction)
- **Y-axis:** Lateral, positive right wing (starboard)
- **Z-axis:** Vertical, positive down (toward ground)

**Right-hand coordinate system**

## Station System (X-axis)

Measured along aircraft centerline from nose apex:

| Station | Location | Description |
|---------|----------|-------------|
| **STA 0** | 0.0 m | Nose apex |
| **STA 5** | 5.0 m | Cockpit forward bulkhead |
| **STA 10** | 10.0 m | Cockpit aft bulkhead |
| **STA 15** | 15.0 m | Center body maximum width |
| **STA 20** | 20.0 m | Forward H₂ tank center |
| **STA 25** | 25.0 m | Wing spar carrythrough |
| **STA 30** | 30.0 m | Aft H₂ tank center |
| **STA 35** | 35.0 m | Aft pressure bulkhead |
| **STA 38.2** | 38.2 m | Tail (aft-most point) |

## Waterline System (Z-axis)

Measured vertically from ground datum:

| Waterline | Height | Description |
|-----------|--------|-------------|
| **WL 0** | 0.0 m | Ground datum |
| **WL 1.8** | 1.8 m | Belly (lowest structural point) |
| **WL 4.0** | 4.0 m | Cabin floor level |
| **WL 8.5** | 8.5 m | Cabin ceiling |
| **WL 10.0** | 10.0 m | Upper deck (H₂ tanks) |
| **WL 14.5** | 14.5 m | Top of fuselage |

## Buttline System (Y-axis)

Measured laterally from centerline:

| Buttline | Distance | Description |
|----------|----------|-------------|
| **BL 0** | 0.0 m | Aircraft centerline |
| **BL 5** | 5.0 m | Fuselage side |
| **BL 10** | 10.0 m | Inner cabin edge |
| **BL 15** | 15.0 m | Main gear attachment |
| **BL 19** | 19.0 m | Center body edge |
| **BL 26** | 26.0 m | Wingtip |

## Critical Reference Points

**Datum Point (0,0,0):** Nose apex at ground level
**CG Reference:** 25% MAC (normal operations)
**Weighing Points:** Main gear trunnions (BL ±15m)
