# PROTO-CKL-003 - H2 Emergency Checklist v1.0

**Prototype ID:** PROTO-CKL-003  
**Version:** 1.0  
**Date:** 2025-10-18  
**Status:** Safety Review

## H2 EMERGENCY CHECKLIST

### H2 LEAK DETECTED
**Memory Items - Immediate Action:**
1. [ ] **H2 MAIN VALVE** - CLOSE
2. [ ] **FUEL CELL SHUTDOWN** - EXECUTE
3. [ ] **BATTERY MODE** - ACTIVATE
4. [ ] **VENTILATION** - MAXIMUM

**Subsequent Actions:**
- [ ] **H2 Leak Location** - IDENTIFY (CAOS DISPLAY)
- [ ] **Fire Detection** - MONITOR
- [ ] **Land ASAP** - PLAN
- [ ] **ATC** - NOTIFY (PAN-PAN or MAYDAY)
- [ ] **QRH** - REFERENCE
- [ ] **Emergency Services** - REQUEST AT DESTINATION
- [ ] **Cabin Crew** - BRIEF
- [ ] **Passengers** - PREPARE

### FUEL CELL FIRE WARNING
**Memory Items:**
1. [ ] **FUEL CELL AFFECTED** - SHUTDOWN
2. [ ] **H2 SUPPLY** - CLOSE
3. [ ] **FIRE EXTINGUISHER** - ACTIVATE
4. [ ] **VENTILATION** - MAXIMUM

**Subsequent Actions:**
- [ ] **Fire Status** - MONITOR
- [ ] **Remaining Fuel Cell** - CHECK STATUS
- [ ] **Battery Power** - ENGAGE
- [ ] **Emergency Descent** - IF REQUIRED
- [ ] **Land Immediately** - NEAREST SUITABLE AIRPORT

### H2 OVERPRESSURE
1. [ ] **H2 VENT VALVE** - OPEN
2. [ ] **Tank Pressure** - MONITOR
3. [ ] **CAOS Diagnostic** - CHECK
4. [ ] **Reduce H2 Flow** - AS REQUIRED
5. [ ] **Land at Nearest Suitable** - PLAN

### FUEL CELL FAILURE (SINGLE)
1. [ ] **Failed Stack** - ISOLATE
2. [ ] **Remaining Stack** - MAXIMUM OUTPUT
3. [ ] **Battery Assist** - ENGAGE
4. [ ] **Non-Essential Systems** - SHED LOAD
5. [ ] **Divert** - NEAREST SUITABLE AIRPORT

### FUEL CELL FAILURE (DUAL)
**Memory Items:**
1. [ ] **BOTH STACKS** - SHUTDOWN
2. [ ] **BATTERY EMERGENCY MODE** - ACTIVATE
3. [ ] **EMERGENCY DESCENT** - INITIATE
4. [ ] **MAYDAY** - DECLARE

**Subsequent Actions:**
- [ ] **Essential Systems Only** - ALL OTHERS OFF
- [ ] **Nearest Airport** - IMMEDIATE LANDING
- [ ] **Emergency Services** - CONFIRM READY
- [ ] **Brace for Landing** - CONSIDER

---

**Critical:** These procedures require immediate memorization  
**Training Required:** Simulator validation mandatory  
**Review Frequency:** Every 6 months
