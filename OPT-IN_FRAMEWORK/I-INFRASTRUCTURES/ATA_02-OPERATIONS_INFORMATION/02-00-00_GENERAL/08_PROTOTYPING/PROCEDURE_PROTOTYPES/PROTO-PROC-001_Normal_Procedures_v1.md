## Overview

This document contains prototype normal operating procedures for the AMPEL360 BWB H2 Hy-E aircraft.

## Pre-Flight Procedures

### 1. External Inspection
1. Check H2 tank visual condition (no frost buildup)
2. Verify fuel cell cooling system connections
3. Inspect BWB control surfaces
4. Check landing gear condition
5. Verify emergency exit accessibility

### 2. Cockpit Preparation
1. Battery Master - ON
2. CAOS System - INITIALIZE
3. Primary Flight Display - CHECK
4. H2 System Panel - VERIFY
5. Fuel Cell Status - CHECK

### 3. Before Engine Start
1. Parking Brake - SET
2. H2 Tank Pressure - VERIFY (350-700 bar)
3. Fuel Cell Temperature - CHECK (60-80Â°C)
4. APU - START (if required)
5. CAOS Advisory - REVIEW

## Engine Start Procedures

### H2 Fuel Cell Startup Sequence
1. H2 Main Valve - OPEN
2. Fuel Cell Stack 1 - INITIALIZE
3. Monitor stack temperature rise
4. Fuel Cell Stack 2 - INITIALIZE
5. Verify electrical output (nominal)
6. CAOS - CONFIRM STARTUP

## Taxi Procedures

1. Release parking brake
2. Verify BWB ground handling characteristics
3. Monitor fuel cell load during taxi
4. Check steering responsiveness
5. CAOS taxi advisory - ACKNOWLEDGE

## Takeoff Procedures

1. Lineup checklist - COMPLETE
2. H2 flow rate - VERIFY
3. Fuel cell power output - MAXIMUM
4. Battery assist - READY
5. V1, VR, V2 - CONFIRM
6. Rotate at VR (verify BWB rotation technique)
7. Positive rate - GEAR UP
