## Overview

This Aircraft Maintenance Manual (AMM) provides comprehensive maintenance instructions for the AMPEL360 BWB H₂ Hy-E Q100 aircraft.

## Structure

The AMM is organized according to ATA iSpec 2200 chapters:
- ATA 02: Operations Information
- ATA 28: Fuel (H₂) Systems
- ATA 52: Doors
- (Additional chapters as developed)

## S1000D Implementation

All maintenance procedures are provided as S1000D Data Modules (DMs) referenced in the AMM_Index.csv and DMC_Register.csv files.

## Usage

1. Locate the required maintenance task in AMM_Index.csv
2. Reference the corresponding DMC from DMC_Register.csv
3. Access the data module XML file for detailed procedures
4. Follow all safety precautions and use specified tools/materials

## H₂ System Safety

Special attention required for hydrogen system maintenance:
- Review DMC-AMP-A-02-00-00-00A-013A-A (Safety Precautions H₂) before any H₂ system work
- Use only approved tools and materials
- Follow pressure system procedures strictly
- Ensure proper ventilation and gas detection
