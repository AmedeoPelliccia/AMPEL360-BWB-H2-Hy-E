## AMPEL360 BWB H₂ Hy-E Q100 INTEGRA

**Document Control:**
- Version: 1.0
- Status: Planning
- Test Date: 2026-Q2 (planned)

---

## 1. Test Objective

Validate BWB-specific weight and balance characteristics, CG envelope, and loading procedures.

---

## 2. Test Scope

### 2.1 CG Envelope
- CG range verification: 15-42% MAC
- Lateral balance limits: ±500 kg maximum imbalance
- Loading distribution validation
- CG calculation system verification

### 2.2 Loading Scenarios
- Maximum passengers + cargo
- Various passenger distributions
- Cargo configurations
- H₂ fuel loading effects
- Emergency scenarios (partial loading)

---

## 3. Test Plan

### 3.1 Equipment
- Precision scales (aircraft weighing)
- Load cells
- CG measurement system
- Loading simulation equipment

### 3.2 Test Sequence
1. Empty weight and CG determination
2. Passenger loading scenarios (10+ configurations)
3. Cargo loading scenarios
4. Combined loading scenarios
5. H₂ fuel loading effects
6. Real-time CG calculation verification

### 3.3 Success Criteria
- CG within 15-42% MAC: PASS/FAIL
- Lateral balance ≤500 kg: PASS/FAIL
- CG calculation accuracy ±0.5%: PASS/FAIL
- Loading procedures executable: PASS/FAIL
