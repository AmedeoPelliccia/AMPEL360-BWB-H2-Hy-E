## Display Content

The EICAS Operations Display shows:

### Fuel Cell Status (Primary)
- 4 Fuel Cell Stacks (FC1, FC2, FC3, FC4)
- Each showing:
  - Power output (kW)
  - Voltage (V)
  - Current (A)
  - Temperature (°C)
  - Status (NORM/CAUTION/WARNING)

### H₂ System Synoptic
- H₂ tank levels (4 tanks)
- Temperatures (cryogenic -253°C)
- Pressures (2.8-3.2 bar)
- Flow rates (kg/hr)
- Leak detection status

### Electrical System
- Bus voltages
- Load distribution
- Generator status
- Battery status

### CAOS Advisory Window
- Current advisories
- Confidence levels
- Quick action buttons

## Color Coding
- Green: Normal operation
- Amber: Caution
- Red: Warning
- White: Advisory/info
- Blue: System mode
