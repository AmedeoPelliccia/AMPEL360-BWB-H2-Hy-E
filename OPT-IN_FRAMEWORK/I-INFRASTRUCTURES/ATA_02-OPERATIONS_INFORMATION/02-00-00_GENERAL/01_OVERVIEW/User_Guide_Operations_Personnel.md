## 9. COMMON TASKS

### 9.1 Calculate Takeoff Performance

**Method 1: Manual (Legacy)**
1. Find 02-40-01 (Takeoff Performance)
2. Locate charts for conditions
3. Read V1, VR, V2
4. Calculate TOD (Takeoff Distance)
5. Verify against runway available

**Method 2: CAOS (Preferred)**
1. CAOS auto-retrieves current data
2. Calculates performance
3. Displays V-speeds + TOD
4. Shows margins
5. Crew verifies and accepts

### 9.2 Plan H2 Fuel

**Method 1: Manual**
1. Calculate trip fuel (02-35-01)
2. Add contingency fuel (5%)
3. Add alternate fuel (02-35-02)
4. Add final reserve (45 min)
5. Add taxi fuel (10 kg)
6. Add extra (if required)

**Method 2: CAOS (Preferred)**
1. CAOS plans complete fuel
2. Uses weather forecast
3. Optimizes for efficiency
4. Shows breakdown
5. Crew verifies and accepts

### 9.3 Respond to H2 Leak

**Immediate Actions (Memory):**
1. H2 LEAK WARNING - VERIFY
2. Affected tank - ISOLATE
3. Ventilation - MAXIMUM
4. QRH - REFER

**Detailed Procedure:**
1. QRH: Immediate action items
2. 02-38-01: Leak response procedure
3. CAOS: Monitors leak evolution
4. Plan diversion if required
5. 02-38-05: Emergency defueling (if needed)
