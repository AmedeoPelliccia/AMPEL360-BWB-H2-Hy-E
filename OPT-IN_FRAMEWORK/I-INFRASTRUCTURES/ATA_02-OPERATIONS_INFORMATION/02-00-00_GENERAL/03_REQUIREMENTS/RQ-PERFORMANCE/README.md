## Overview

Performance requirements define the operational capabilities and efficiency targets for the AMPEL360 aircraft.

## Key Performance Targets

- **Range:** 3500 NM
- **Cruise Speed:** Mach 0.82
- **Service Ceiling:** FL450
- **Takeoff Distance:** 2500m at MTOW
- **Landing Distance:** 2000m at MLW
- **Climb Gradient:** 3% minimum
- **Fuel Efficiency:** 30% better than baseline
- **Fuel Cell Efficiency:** 55-60%
- **Crosswind:** 35 knots
- **CAOS Optimization:** 5% improvement
