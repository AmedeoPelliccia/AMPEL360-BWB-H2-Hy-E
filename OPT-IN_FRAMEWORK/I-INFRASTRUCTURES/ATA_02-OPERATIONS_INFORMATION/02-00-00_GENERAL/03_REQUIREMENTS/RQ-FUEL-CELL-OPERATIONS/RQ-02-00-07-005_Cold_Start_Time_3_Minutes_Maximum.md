# RQ-02-00-07-005: Cold Start Time 3 Minutes Maximum

## Requirement Details

**ID:** RQ-02-00-07-005  
**Category:** Fuel Cell Ops  
**Subcategory:** Start  
**Title:** Cold Start Time 3 Minutes Maximum  
**Priority:** High  
**Status:** Approved

## Description

The PEM fuel cell propulsion system shall achieve cold start time 3 minutes maximum for reliable electric power generation.

## Rationale

This fuel cell requirement ensures reliable, efficient electric power generation from hydrogen fuel.
