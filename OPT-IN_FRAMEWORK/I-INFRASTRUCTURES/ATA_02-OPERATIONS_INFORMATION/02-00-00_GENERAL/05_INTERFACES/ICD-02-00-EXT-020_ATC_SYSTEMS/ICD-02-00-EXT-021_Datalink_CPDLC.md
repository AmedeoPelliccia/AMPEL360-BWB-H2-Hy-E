# ICD-02-00-EXT-021: Datalink CPDLC
**Version:** 1.0.0 | **Status:** Active

## Interface Description
Controller-Pilot Data Link Communications (CPDLC) implementation for ATC coordination.

## CPDLC Capabilities
- Standard ICAO CPDLC messages
- Route amendments
- Altitude clearances
- Speed restrictions
- Emergency messaging

## Data Link Systems
- **Primary**: FANS 1/A (ACARS)
- **Secondary**: ATN B2 (VDL Mode 2)
- **Coverage**: Oceanic + domestic

## H2-Specific Messaging
- H2 emergency notifications
- Diversion requirements (H2-capable airports)
- Special handling requests

**RQ-ICD-02-EXT-210:** CPDLC FANS 1/A compliant  
**RQ-ICD-02-EXT-211:** H2 emergency messaging capability
