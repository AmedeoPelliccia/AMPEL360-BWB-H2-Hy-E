# ICD-02-00-EXT-020: Flight Plan Interface
**Version:** 1.0.0 | **Status:** Active

## Interface Description
Flight planning data exchange between AMPEL360 operations and ATC systems.

## Flight Plan Elements
- Aircraft type: AMPEL360 (new type code)
- Equipment codes: H2 fuel, fuel cell propulsion
- Performance category: Medium jet equivalent
- Special handling: H2 diversion airports required

## ICAO Flight Plan
- **Equipment**: SBDEG/H (standard plus H2)
- **Surveillance**: ADSB OUT (1090ES)
- **Wake Turbulence**: Medium (similar to A320 family)

## Special Requirements
- Alternate airports must be H2-capable
- ETOPS equivalent for H2 infrastructure
- Emergency diversion airports identified

**RQ-ICD-02-EXT-200:** Flight plan H2-capable alternates  
**RQ-ICD-02-EXT-201:** ATC notification of H2 fuel type
