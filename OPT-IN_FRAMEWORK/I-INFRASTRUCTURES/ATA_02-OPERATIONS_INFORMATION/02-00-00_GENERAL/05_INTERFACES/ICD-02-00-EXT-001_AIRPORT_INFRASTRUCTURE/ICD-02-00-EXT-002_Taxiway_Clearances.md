# ICD-02-00-EXT-002: Taxiway Clearances
**Version:** 1.0.0 | **Status:** Active

## Interface Description
Taxiway infrastructure requirements for BWB ground operations.

## Taxiway Dimensions
- **Width**: 60m minimum (Code F)
- **Clearance**: 40m wingtip to obstacle
- **Strength**: PCN 80/R/B/W/T minimum
- **Turn Radius**: 50m centerline minimum

## Special Considerations
- BWB 80m wingspan requires Code F facilities
- Wide-stance landing gear distribution
- Ground maneuvering cameras required
- Marshaller training for BWB dimensions

**RQ-ICD-02-EXT-010:** Taxiway Code F compatible  
**RQ-ICD-02-EXT-011:** PCN ≥80 pavement strength
