# ICD-02-00-EXT-003: Runway Requirements
**Version:** 1.0.0 | **Status:** Active

## Interface Description
Runway requirements for AMPEL360 takeoff and landing operations.

## Runway Specifications
- **Length**: 2,500m minimum (MTOW, ISA)
- **Width**: 45m minimum (Code 4E)
- **Strength**: PCN 80/R/B/W/T
- **Surface**: Grooved or porous friction coating preferred

## Performance Considerations
- Takeoff distance: 2,200m (MTOW, ISA, dry)
- Landing distance: 1,800m (MLW, ISA, dry)
- Crosswind limit: 35 knots (BWB stability)
- Wet runway: +15% distance

**RQ-ICD-02-EXT-020:** Runway length ≥2,500m  
**RQ-ICD-02-EXT-021:** Runway width ≥45m
