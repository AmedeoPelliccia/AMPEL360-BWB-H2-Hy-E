# ICD-02-00-EXT-001: Gate Requirements
**Version:** 1.0.0 | **Status:** Active

## Interface Description
Airport gate infrastructure requirements for AMPEL360 BWB operations.

## Gate Dimensions
- **Clearance Width**: 90m minimum (80m wingspan + 5m each side)
- **Clearance Depth**: 85m minimum
- **Vertical Clearance**: 25m minimum
- **Turning Radius**: 45m minimum at gate

## Ground Services
- GPU: 115V 400Hz, 28VDC
- Air conditioning: Compatible with BWB cabin
- Passenger boarding: Multiple door capability
- Cargo loading: Wide-body compatible loaders

## H2-Specific Requirements
- H2 refueling point (50m from terminal minimum)
- H2 leak detection in refueling area
- Fire suppression foam capability
- H2-trained ground crew available

**RQ-ICD-02-EXT-001:** Gate width ≥90m  
**RQ-ICD-02-EXT-002:** H2 refueling 50m from terminal
