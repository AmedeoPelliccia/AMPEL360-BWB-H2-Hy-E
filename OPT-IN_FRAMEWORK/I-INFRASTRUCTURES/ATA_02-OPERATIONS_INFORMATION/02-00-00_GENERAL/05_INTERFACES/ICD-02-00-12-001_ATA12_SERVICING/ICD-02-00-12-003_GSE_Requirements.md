# ICD-02-00-12-003: Ground Support Equipment Requirements
**Version:** 1.0.0 | **Status:** Active

## Interface Description
Ground support equipment specifications and requirements for AMPEL360 servicing operations.

## Required GSE

### H2 Servicing
- **H2 Refueling Truck**: ISO 19881 compliant, 10,000 kg capacity
- **H2 Leak Detectors**: Portable, 0-100% LEL range
- **Cryogenic Transfer Hoses**: -253°C rated
- **Emergency Defueling Equipment**: Portable pump system

### General Servicing
- **GPU**: 115V 400Hz, 28VDC capability
- **Air Start Unit**: For fuel cell startup
- **Tow Bar**: BWB-specific, 150,000 kg capacity
- **Jacks/Stands**: Wide stance for BWB configuration

## Interface Requirements
**RQ-ICD-02-12-020:** All H2 GSE ISO 19881 certified  
**RQ-ICD-02-12-021:** GSE operators H2-safety trained  
**RQ-ICD-02-12-022:** GSE maintenance records current
