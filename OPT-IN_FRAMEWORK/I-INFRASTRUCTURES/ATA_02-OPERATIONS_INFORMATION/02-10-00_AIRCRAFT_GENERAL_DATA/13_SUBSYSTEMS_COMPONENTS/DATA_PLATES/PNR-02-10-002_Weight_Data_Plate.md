# PNR-02-10-002 - Weight Data Plate

**Part Number:** PNR-02-10-002  
**Type:** Component  
**Manufacturer:** AMPEL360  
**Status:** Active

## Description

Weight and balance data plate providing certified empty weight, center of gravity, and configuration information for the AMPEL360 BWB H2 Hy-E Q100 INTEGRA aircraft.

## Specifications

- **Material:** Stainless steel 316
- **Dimensions:** 150mm × 100mm × 2mm
- **Mounting:** Adhesive-backed
- **Location:** Cockpit wall, left side
- **Finish:** Laser-engraved

## Data Fields

- Aircraft Serial Number
- Registration Number
- Empty Weight (kg)
- Empty Weight CG (% MAC)
- Date of Last Weighing
- Weighing Agency
- Configuration Code
- Approval Signature

## Maintenance

- Update required after major modifications
- Re-weighing interval: Per maintenance schedule
- Replacement: When illegible or damaged
