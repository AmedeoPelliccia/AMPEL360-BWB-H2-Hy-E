# PNR-02-10-102 - Station Marking Decal Set

**Part Number:** PNR-02-10-102  
**Type:** Consumable  
**Manufacturer:** AeroMark  
**Status:** Active

## Description

Complete set of fuselage station marking decals for the AMPEL360 BWB H2 Hy-E aircraft. Provides reference points for maintenance, inspection, and structural repairs.

## Specifications

- **Material:** Polyester film with permanent adhesive
- **Size:** 50mm × 30mm per decal
- **Color:** Black on white, high contrast
- **Durability:** 10 years minimum
- **Temperature Range:** -55°C to +125°C
- **UV Resistance:** Excellent

## Kit Contents

- Station markers: 0, 50, 100, 150, ... 1000 (increments of 50mm)
- Quantity: 21 decals per set
- Waterline reference markers: WL 0 to WL 500
- Buttock line reference markers: BL 0 (centerline) to BL ±500

## Application Locations

### Fuselage Stations (FS)
- Marked on left side of fuselage
- Measured from aircraft datum point
- Interval: 50mm for accessible areas

### Waterlines (WL)
- Marked on forward and aft bulkheads
- Measured from reference plane

### Buttock Lines (BL)
- Marked on floor panels
- Measured from aircraft centerline

## Installation Instructions

1. Clean surface with isopropyl alcohol
2. Verify station location with measurement tools
3. Apply decal, remove air bubbles
4. Press firmly for 30 seconds
5. Allow 24 hours for adhesive cure
