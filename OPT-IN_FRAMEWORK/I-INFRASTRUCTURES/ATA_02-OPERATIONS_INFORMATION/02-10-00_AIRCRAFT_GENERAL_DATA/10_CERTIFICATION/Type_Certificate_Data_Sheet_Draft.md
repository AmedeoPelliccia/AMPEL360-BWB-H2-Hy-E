## XI. Notes

### Note 1: Blended Wing Body Configuration
This aircraft features an unconventional Blended Wing Body (BWB) design certified under Special Condition SC-BWB-001. Specific training and operational procedures are required due to unique flight characteristics.

### Note 2: Hydrogen Fuel System
This aircraft is powered exclusively by hydrogen fuel cells. Operation requires:
- Certified hydrogen refueling facilities
- Trained ground crew (H₂ safety certification)
- Special fire/rescue procedures at airports
- Compliance with ISO 19881 and aviation adaptations

### Note 3: Center of Gravity Management
This aircraft features an active CG control system using hydrogen fuel positioning. The wide CG range (15-42% MAC) is certified under Special Condition SC-BWB-002.

### Note 4: ICAO Code E Compliance
This aircraft meets ICAO Annex 14 Code E requirements:
- Wingspan: 65.0 m (Code E limit: 65 m)
- Requires Code E airport infrastructure
- Reference field length: 2,400 m

### Note 5: Environmental Features
- **Emissions:** Zero CO₂, zero NOx, zero particulates from propulsion
- **Noise:** 65 dB approach (75% quieter than conventional aircraft)
- **Optional CO₂ Capture:** Net carbon-negative operation possible
