**Status:** Active

---

## Purpose

This directory contains tables, catalogs, schemas, software design specifications (SDS), training materials, and metadata for ATA 03 Ground Support Equipment systems.

---

## Structure

Concrete systems under this origin block are structured according to their design and implementation needs, not following the mandatory 14-folder lifecycle pattern.

Each subsystem `03-90-YY_*` has its own internal design-driven structure.
