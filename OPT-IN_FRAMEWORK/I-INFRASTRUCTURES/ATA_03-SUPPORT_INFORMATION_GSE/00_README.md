# ATA 03 - Support Information GSE

## Overview
This chapter contains specifications, manuals, and certification documentation for all Ground Support Equipment (GSE) used with the AMPEL360 aircraft.

## Structure

### 01-interfaces
Interface specifications between aircraft and GSE

### 02-gse_manuals
Operating and maintenance manuals for all GSE

### 03-software_firmware
Software and firmware documentation for electronic GSE

### 04-acceptance_and_certification
Acceptance criteria and certification documentation

### 05-safety
Safety procedures and protocols for GSE operations

### 06-training
Training materials and requirements for GSE operators

### 07-maintenance_and_calibration
Maintenance schedules and calibration procedures

### 08-spares
Spare parts catalogs and management

### 09-registry
Registry of all approved GSE

## Governance
All GSE must be certified and approved for use with the AMPEL360 aircraft, with particular attention to hydrogen safety requirements.
