# AMPEL360 Validators

This package provides validation utilities for the AMPEL360 project.

## Overview

The validators package contains tools for validating:
- Documentation structure and metadata
- Configuration files
- Data formats and schemas
- Cross-reference integrity

## Installation

The package can be installed in development mode:

```bash
pip install -e tools/validators
```

Or as a regular package:

```bash
pip install tools/validators
```

## Usage

The validators are typically used as part of the CI/CD pipeline to ensure
quality and consistency across the AMPEL360 project.

## Structure

- `__init__.py` - Package initialization
- `pyproject.toml` - Package metadata and configuration

## Development

To install with development dependencies:

```bash
pip install -e "tools/validators[dev]"
```
